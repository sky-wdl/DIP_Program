# DIP_Program
上传我的期末大作业，做代码记录，本程序可做一些简单的图像处理效果演示。

main为主程序，实现程序的功能。miangui为用pyqt5绘制的gui文件，实现程序的gui界面效果。

本程序依赖Qt5，请务必保证使用pip安装了qt5，qt6是否兼容未测试。
